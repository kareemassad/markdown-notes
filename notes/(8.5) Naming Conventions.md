# (8.5) Naming Conventions 

1) Give Variables meaningful names!
2) Variable names cannot begin with a number.
3) Variable names cannot contain spaces.
4) Only `$` & `_` are valid aside form letters and numbers.
5) Use camelCasing
```js
var userName = "Kate";
```
