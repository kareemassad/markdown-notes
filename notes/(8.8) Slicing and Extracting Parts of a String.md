# (8.8) Slicing and Extracting Parts of a String 

### slice(x,y)

Let's say you only want the first character of this string
```js
var name = "Kareem";
name.slice(0,1)
```
* This says that you want to slice string `name` from position 0 to 1 but not including 1. 

* Count starts from 0.
