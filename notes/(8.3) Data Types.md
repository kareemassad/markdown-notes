# (8.3) Data Types 

###### DATE: 11/8/2019

### List of Data Types:

1) String
- A string of characters 

2) Boolean
- Can be TRUE or FALSE

3) Number



### The typeof() function

This function indicates the type of data structure of the argument.
```js
typeof(23);
= "number"
```
```js
typeof("Kareem");
= "string"
```
```js
typeof(true);
= "boolean"
```
