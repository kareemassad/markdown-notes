# (8.18) Parameters and Arguments

Using parameters
```js
function functionName(parameters){
  //code here
}
```

Example 1: 
```js
//creating the function
function getMilk(bottles){
  var cost = bottles * 1.5;
  //do something with cost
}
//calling the function
getMilk(2);
```
