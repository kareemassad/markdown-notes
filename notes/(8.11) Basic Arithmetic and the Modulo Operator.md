# (8.11) Basic Arithmetic and the Modulo Operator

Ex: 

```js
var e = 9 % 6 // = 3
```
6 goes into 9 only once and the remainder is 3

#### USES
Can be used to determine if a number is odd or even

### Arithmetic

Same as math, * and / have precedence over + and - unless you use ( ).
